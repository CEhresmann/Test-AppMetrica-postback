# Обработка Postback от AppMetrica

Этот проект предоставляет Go-сервер для приема и обработки postback-запросов от AppMetrica. Он извлекает информацию о пути запроса и всех параметрах, а затем отображает их в виде HTML-документа.

## Локальный запуск

 **Запустите сервер:** Откройте терминал в директории с файлом `main.go` и выполните команду:
    ```bash
    go run main.go
    ```
    Сервер запустится и будет слушать на порту `8080`.
4.  **Тестирование:** Отправьте postback-запрос на `http://localhost:8080/` с нужными параметрами. Например:
    ```
    http://localhost:8080/?product=123&platform=android&time=1678886400&device=abcdef12345
    ```
    Вы увидите HTML-страницу с деталями запроса.

## **Важное примечание:** 
Для приема postback вам потребуется отдельный сервер или сервис для запуска вашего Go-приложения. После развертывания на таком сервере вы получите URL, на который сможете настроить postback от AppMetrica.
