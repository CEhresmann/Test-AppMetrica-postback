# Обработка Postback от AppMetrica

Этот проект предоставляет Go-сервер для приема и обработки postback-запросов от AppMetrica. Он извлекает информацию о пути запроса и всех параметрах, а затем отображает их.

## Хостинг

- **Бэкенд (Amvera):** [https://testpostback-timofey3498.amvera.io/](https://testpostback-timofey3498.amvera.io/)

## Как это работает

1.  **Прием данных:** Бэкенд слушает эндпоинт `/postback`.
2.  **Просмотр данных:** Эндпоинт `/view` возвращает HTML с последними полученными данными.
3.  **Фронтенд:** Статическая страница на GitHub Pages опрашивает бэкенд каждые 5 секунд и отображает актуальную информацию.

## Локальный запуск

1.  **Запустите сервер:** Откройте терминал в директории с файлом `main.go` и выполните команду:
    ```bash
    go run main.go
    ```
    Сервер запустится и будет слушать на порту `8080`.
2.  **Тестирование:** Отправьте postback-запрос на `http://localhost:8080/postback` с нужными параметрами. Например:
    ```
    http://localhost:8080/postback?product=123&platform=android&time=1678886400&device=abcdef12345
    ```
3.  **Просмотр:** Откройте `frontend/index.html` в браузере или перейдите на `http://localhost:8080/`.

## Настройка AppMetrica

В интерфейсе AppMetrica укажите следующий URL для Postback:
`https://testpostback-timofey3498.amvera.io/postback?параметры...`
